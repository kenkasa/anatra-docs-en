Usage of fanatra

Introduction

fanatra stands for “Fortran ANATRA” and, as its name suggests, is a collection of analysis programs written in Fortran. This chapter describes how to use fanatra. Since fanatra performs analyses without invoking VMD, it can also be used on computing systems where the use of VMD on compute nodes is difficult. A list of available analysis modes can be displayed by executing fanatra -h.

$ fanatra -h
Available analysis in Fortran ANATRA
histogram               (hist)   : Histogram Analysis
potential_of_mean_force (pmf)    : Free-energy Profile Analysis
restricted_radial       (rrpmf)  : Restricted Radial Analysis
extract_frame           (extf)   : Extract Frames Using CV
random_frame            (randf)  : Randomly Extract Frames
state_define            (sdef)   : Define States From Time-series Data
moving_average          (movave) : Moving Average Analysis
average_function        (avef)   : Function Average Analysis
bootstrap_prepper       (boot)   : Bootstrap Sample Generator
return_prob             (rp)     : Returning Probability Analysis
rprate                  (rprate) : Rate Constant Analysis based on RP theory
iepdyn                  (iepdyn) : Integral-equation Formalism of Population Dynamics

Among the above, entries such as histogram are analysis modes, and their abbreviated names (e.g., hist) can also be used.

While anatra performs analyses by specifying parameters directly on the command line, fanatra uses a separate parameter file. For example,

$ fanatra hist control_file

The parameter file must be written in the Fortran namelist format.

In addition, the format and contents of the parameter file for each analysis mode can be displayed together with a brief explanation and a sample input by executing fanatra <analysis_mode> -h. Therefore, the basic usage of each analysis mode can be understood from the help message. An example for histogram is shown below.

$ fanatra histogram -h

Most parameter files in fanatra are based on the Fortran namelist format. Parameters are specified in the input_param, output_param, and option_param sections.

The input_param and output_param sections, which are common to all analysis modes, are described below.

Format of Time-series Data

The format of the time-series data is as follows.

  • 1st column: Time axis

  • 2nd column: First time-series data

  • 3rd column: Second time-series data

The first column is ignored in all analysis modes, and the second column is treated as the first data column.

Format of Statistical Weight Data

The format of the statistical weight data is as follows.

  • 1st column: Time axis

  • 2nd column: Statistical weight

The first column is ignored in all analysis modes, and only the second column is used.

The number of rows in the statistical weight data file must be identical to that of the corresponding time-series data file.

Histogram (histogram, hist)

In this analysis mode, the distribution function \(P\left(x\right)\) is calculated from a given time-series data set \(\xi\left(t\right)\).

(46)\[\begin{align} P\left(x\right) = \dfrac{1}{N}\Braket{\delta\left(x - \xi\right)} \end{align}\]

where \(N\) is the number of data points. The above expression satisfies the normalization condition

(47)\[\begin{align} \int_{-\infty}^{\infty} dx P\left(x\right) = 1 \end{align}\]

Potential of Mean Force (potential_of_mean_force, pmf)

In this analysis mode, the free-energy profile \(W\left(\boldsymbol{\zeta}\right)\) along the reaction coordinate \(\boldsymbol{\zeta}\) is calculated from the time-series data of the reaction coordinate, \(\boldsymbol{\xi}\left(t\right)\). The function \(W\left(\boldsymbol{\zeta}\right)\) is also referred to as the potential of mean force (PMF).

Theoretical Background

Let \({\bf X}\) denote the configuration of the system and \(U\left({\bf X}\right)\) its potential energy. The configurational integral \(Z\) is given by

(49)\[\begin{align} Z =\int d{\bf X}\,\exp\left(-\beta U\left({\bf X}\right)\right) \end{align}\]

where \(\beta\) is the inverse temperature.

Similarly, the partition function corresponding to the state in which the system is trapped within the infinitesimal region \(\boldsymbol{\zeta} - \boldsymbol{\zeta}+\Delta \boldsymbol{\zeta}\) in the reaction coordinate space (\(\boldsymbol{\zeta}\left({\bf X}\right)\)) is

(50)\[\begin{align} Z\left(\boldsymbol{\zeta}\right) =\Delta\boldsymbol{\zeta}\int d{\bf X}\,\delta\left(\boldsymbol{\zeta}-\boldsymbol{\xi}\left({\bf X}\right)\right)\exp\left(-\beta U\left({\bf X}\right)\right) \end{align}\]

Defining the probability density corresponding to \(\boldsymbol{\zeta}\) as

(51)\[\begin{align} \rho\left(\boldsymbol{\zeta}\right) = \Braket{\delta\left(\boldsymbol{\zeta} - \boldsymbol{\xi}\left({\bf X}\right)\right)} = \dfrac{1}{Z}\int d{\bf X}\,\delta\left(\boldsymbol{\zeta}-\boldsymbol{\xi}\left({\bf X}\right)\right)\exp\left(-\beta U\left({\bf X}\right)\right) \end{align}\]

we obtain

(52)\[\begin{align} \dfrac{Z\left(\boldsymbol{\zeta}\right)}{Z} =\rho\left(\boldsymbol{\zeta}\right)\Delta\boldsymbol{\zeta} \end{align}\]

Therefore, the free-energy difference \(\Delta F\left(\boldsymbol{\zeta}\right)\) between the unrestricted state and the state confined to the region \(\boldsymbol{\zeta}-\boldsymbol{\zeta} + \Delta \boldsymbol{\zeta}\) is expressed as

(53)\[\begin{align} \Delta F\left(\boldsymbol{\zeta}\right) =-\dfrac{1}{\beta}\log\dfrac{Z\left(\boldsymbol{\zeta}\right)}{Z} = -\dfrac{1}{\beta}\log \left(\rho\left(\boldsymbol{\zeta}\right)\Delta \boldsymbol{\zeta}\right) \end{align}\]

The free energy relative to a reference point \(\boldsymbol{\zeta}_{0}\) is

(54)\[\begin{align} W\left(\boldsymbol{\zeta}\right) =\Delta F\left(\boldsymbol{\zeta}\right)-\Delta F\left(\boldsymbol{\zeta}_{0}\right)\notag\\ =-\dfrac{1}{\beta}\log\dfrac{\rho\left(\boldsymbol{\zeta}\right)}{\rho\left(\boldsymbol{\zeta}_{0}\right)} \label{pmf:pmf} \end{align}\]

This quantity \(W\) is called the potential of mean force. The choice of \(\boldsymbol{\zeta}_{0}\) is arbitrary, but in most cases it is chosen such that the minimum value of \(W\left(\boldsymbol{\zeta}\right)\) becomes zero.

By calculating \(W\left(\boldsymbol{\zeta}\right)\), it becomes possible to discuss the change in thermodynamic stability as a function of \(\boldsymbol{\zeta}\).

Equation \eqref{pmf:pmf} can also be rewritten by separating the \(\boldsymbol{\zeta}\)-dependent part and a constant \(C\):

(55)\[\begin{align} W\left(\boldsymbol{\zeta}\right) =-\dfrac{1}{\beta}\log\int d{\bf X}\,\delta\left(\boldsymbol{\zeta}-\boldsymbol{\xi}\left({\bf X}\right)\right)\exp\left(-\beta U\left({\bf X}\right)\right)+C \label{pmf:pmf2} \end{align}\]

where

(56)\[\begin{align} C = \dfrac{1}{\beta}\log\int d{\bf X}\,\delta\left(\boldsymbol{\zeta}_{0}-\boldsymbol{\xi}\left({\bf X}\right)\right)\exp\left(-\beta U\left({\bf X}\right)\right) \end{align}\]

&option_param

  • ndim

    • Default: None

    • Example: ndim = 2

    Specifies the number of dimensions of the Collective Variable (CV).

    For example, ndim = 2 calculates a two-dimensional PMF, while ndim = 3 calculates a three-dimensional PMF.

  • xyzcol

    • Default: 1

    • Example: xyzcol = 1 2

    Specifies the column indices corresponding to each CV in the CV data file.

    The first column is ignored because it is treated as the time or step index.

    For example, if the data are stored as

    time  cv1  cv2  cv3

    then

    xyzcol = 1 2

    calculates a two-dimensional PMF using cv1 and cv2.

  • ng3

    • Default: 1 1 1

    • Example: ng3 = 10 20

    Specifies the number of grid points for each CV direction.

    When ndim = N, the grid numbers for each dimension should be specified sequentially in ng3.

  • del

    • Default: 1.0 1.0 1.0

    • Example: del = 0.1d0 0.2d0

    Specifies the grid spacing for each CV direction.

    For the \(i\)-th CV, if the grid spacing is denoted by \(\Delta_i\), the distance between adjacent grid points is \(\Delta_i\).

  • origin

    • Default: 0.0 0.0 0.0

    • Example: origin = 0.0d0 1.0d0

    Specifies the grid origin for each CV direction.

    For the \(k\)-th CV, let the origin be \(x_k^{(0)}\) and the grid spacing be \(\Delta_k\). Then the position of the \(i\)-th grid point is given by

    • Example: spline_resolution = 8

    Specifies the resolution used for spline interpolation.

    Larger values generate a finer interpolation grid.

Restricted Radial Potential of Mean Force (restricted_radial, rrpmf)

This analysis mode calculates the free energy profile along the radial direction under a condition defined by one or more reaction coordinates (collective variables).

Using this profile, the free energy change associated with the binding of a host molecule and a guest molecule (binding free energy) can also be evaluated.

Theoretical Background

Let \({\bf X}\) denote the configuration of the system. The intermolecular distance between the host and guest molecules and the reaction coordinates defining the binding state are denoted by \(d\left({\bf X}\right)\) and \(\boldsymbol{\xi}\left({\bf X}\right)\), respectively.

The potential of mean force along the intermolecular distance is defined as

(59)\[\begin{align} w\left(r\right) =-\dfrac{1}{\beta}\log\left[\int d{\bf X}\,\dfrac{\delta\left(r-d\left({\bf X}\right)\right)}{4\pi r^{2}}\exp\left(-\beta U\left({\bf X}\right)\right)\right]+C \end{align}\]

Furthermore, let \(\boldsymbol{\Upsilon}\) be the region in the reaction coordinate space corresponding to the bound state, and introduce the characteristic function \(\Theta\left(\boldsymbol{\xi}\right)\) defined as

(60)\[\begin{align} \Theta_{\mathrm{B}}\left(\boldsymbol{\xi}\right) =\begin{cases} 1 & \boldsymbol{\xi}\in\boldsymbol{\Upsilon}\\ 0 & \boldsymbol{\xi}\notin\boldsymbol{\Upsilon} \end{cases} \end{align}\]

Using this function, the PMF for configurations belonging to the bound state is expressed as

(61)\[\begin{align} w_{\mathrm{B}}\left(r\right) =-\dfrac{1}{\beta}\log\left[\int d{\bf X}\,\dfrac{\delta\left(r-d\left({\bf X}\right)\right)}{4\pi r^{2}}\Theta_{\mathrm{B}}\left(\boldsymbol{\xi}\left({\bf X}\right)\right)\exp\left(-\beta U\left({\bf X}\right)\right)\right]+C \end{align}\]

The constant \(C\) appearing in both \(w\left(r\right)\) and \(w_{\mathrm{B}}\left(r\right)\) is taken to be the same value, and is chosen such that the minimum value of \(w_{\mathrm{B}}\left(r\right)\) becomes zero.

Since \(w\left(r\right)\) becomes constant when \(r\) is sufficiently large, let this constant be \(\Delta w\), and define the region satisfying \(w\left(r\right)=\Delta w\) as \(r_{0}\leq r\leq r_{1}\). The choice of \(r_{0}\) and \(r_{1}\) is arbitrary as long as this condition is satisfied. This point will be discussed later.

The probability ratio between the unbound state U, where the two molecules are separated by \(r_{0}\leq r \leq r_{1}\), and the bound state B is

(62)\[\begin{align} \dfrac{{\displaystyle \int_{\boldsymbol{\Upsilon}}dr\,\exp\left(-\beta w_{\mathrm{B}}\left(r\right)\right)}}{{\displaystyle \int_{r_{0}}^{r_{1}}dr\,\exp\left(-\beta w\left(r\right)\right)}} \end{align}\]

Therefore, the free energy change \(\Delta G_{\mathrm{PMF}}\) associated with the transition from U to B is

(63)\[\begin{align} \Delta G_{\mathrm{PMF}} =-\dfrac{1}{\beta}\log\dfrac{{\displaystyle \int_{\boldsymbol{\Upsilon}}dr\,4\pi r^{2}\exp\left(-\beta w_{\mathrm{B}}\left(r\right)\right)}}{{\displaystyle \int_{r_{0}}^{r_{1}}dr\,4\pi r^{2}\exp\left(-\beta w\left(r\right)\right)}} \end{align}\]

Since \(w\left(r\right)=\Delta w\) within the range \(r_{0}\leq r\leq r_{1}\), this equation can be rewritten as

(64)\[\begin{align} \Delta G_{\mathrm{PMF}} =-\Delta w-\dfrac{1}{\beta}\log\dfrac{V_{\mathrm{B}}}{V_{\mathrm{U}}} \label{pmf:dGpmf} \end{align}\]

where

(65)\[\begin{align} V_{\mathrm{B}} =\int_{\boldsymbol{\Upsilon}}dr\,4\pi r^{2}\exp\left(-\beta w_{\mathrm{B}}\left(r\right)\right)\\ V_{\mathrm{U}} =\int_{r_{0}}^{r_{1}}dr\,4\pi r^{2} \end{align}\]

The second term in Eq. \eqref{pmf:dGpmf} represents the free energy change associated with the transition of the guest molecule from a concentration of \(1/V_{\mathrm{U}}\) to a concentration of \(1/V_{\mathrm{B}}\).

The experimentally measured binding free energy is the free energy change associated with binding a guest molecule at the standard-state concentration (typically \(c^{\circ} = 1~\mathrm{mol~L^{-1}}\)), and is therefore different from \(\Delta G_{\mathrm{PMF}}\).

Let \(V^{\circ}\) be the volume corresponding to a concentration of \(c^{\circ}\) for a single guest molecule. Then \(V^{\circ} = 1661\) Angstrom\(^{3}\).

The free energy change \(\Delta G^{\circ}_{\mathrm{corr}}\) associated with the transition from concentration \(c^{\circ}=1/V^{\circ}\) to \(1/V_{\mathrm{U}}\) is

(66)\[\begin{align} \Delta G^{\circ}_{\mathrm{corr}} = - \dfrac{1}{\beta} \log \dfrac{V_{\mathrm{U}}}{V^{\circ}} \label{pmf:dGcorr} \end{align}\]

Thus, by decomposing the transition from \(1/V^{\circ}\) to \(1/V_{\mathrm{B}}\) into the two steps,

(67)\[\begin{align} 1/V^{\circ} \to 1/V_{\mathrm{U}} \end{align}\]

and

(68)\[\begin{align} 1/V_{\mathrm{U}} \to 1/V_{\mathrm{B}} \end{align}\]

we obtain

(69)\[\begin{align} \Delta G^{\circ} = \Delta G_{\mathrm{PMF}} + \Delta G_{\mathrm{corr}}^{\circ} \end{align}\]

Substituting Eqs. \eqref{pmf:dGpmf} and \eqref{pmf:dGcorr} into the above equation yields

(70)\[\begin{align} \Delta G^{\circ} = - \Delta w - \dfrac{1}{\beta} \log \dfrac{V_{\mathrm{B}}}{V^{\circ}} \end{align}\]

Since this expression does not contain \(V_{\mathrm{U}}\), the value of \(\Delta G^{\circ}\) is independent of the choice of \(r_{0}\) and \(r_{1}\).

Format of the Input File (Time-Series Data)

  • 1st column: Time

  • 2nd to \((N+1)\)-th columns: CV values

  • Last column: Intermolecular distance

The first column is ignored, and the second column is treated as the first data column.

If the intermolecular distance is the only CV, i.e.,

  • 1st column: Time

  • 2nd column: Intermolecular distance

then simply specify

only_r = .true.

in &option_param.

&option_param

  • calcfe

    • Default: .true.

    • Example: calcfe = .true.

    Specifies whether the standard binding free energy should be calculated.

    If .true., the standard binding free energy is calculated. If .false., only the PMF profile is generated.

  • ndim

    • Default: None

    • Example: ndim = 1

    Specifies the number of CV dimensions.

    Currently, both one-dimensional and multidimensional PMF calculations are supported.

  • only_r

    • Default: .false.

    • Example: only_r = .false.

    Set this option to .true. when the intermolecular distance is the only CV and no additional CVs are present.

  • temperature

    • Default: 298.0

    • Example: temperature = 300.0

    Specifies the temperature \(T\) [K] used for the free energy calculation.

  • dr

    • Default: None

    • Example: dr = 0.1d0

    Specifies the grid spacing along the distance coordinate.

    This value is used as the grid interval \(\Delta r\).

  • ngrid

    • Default: None

    • Example: ngrid = 1000

    Specifies the number of grid points along the distance coordinate.

    The PMF is evaluated on ngrid grid points.

  • nsta

    • Default: 1

    • Example: nsta = 100

    Specifies the frame number from which the analysis starts.

    Frames before nsta are excluded from the analysis.

  • use_bootstrap

    • Default: .false.

    • Example: use_bootstrap = .true.

    Specifies whether bootstrap error analysis is performed.

    If .true., bootstrap analysis is carried out according to the settings in &bootopt_param.

  • vol0

    • Default: 1661.0d0

    • Example: vol0 = 1661.0d0

    Specifies the standard-state volume \(V^\circ\).

    The default value corresponds to the volume for a concentration of 1 mol/L in Angstrom units (1661 Angstrom\(^3\)).

  • urange

    • Default: None

    • Example: urange = 25.0 30.0

    Specifies the distance range regarded as the unbound state.

    The first value represents the lower bound, and the second value represents the upper bound.

  • bound_range

    • Default: None

    • Example:

      react_range = 0.0 10.0
              5.0 20.0

    Specifies the CV range corresponding to the bound state.

    For each line, the first value represents the lower bound and the second value represents the upper bound.

    By specifying ndim ranges, a multidimensional bound region can be defined.

&option_param

  • xsta

    • Default: 0.0d0

    • Example: xsta = 0.0d0

    Specifies the starting position of the grid.

    If the position of the \(i\)-th grid point is denoted by \(x_i\),

    (71)\[\begin{align} x_i = x_0 + (i-1)\Delta x \end{align}\]

    where \(x_0\) is the starting position.

  • nregion

    • Default: 1

    • Example: nregion = 2

    Specifies the number of regions to which different moving-average conditions are applied.

    The number of values given to npoint must match this number of regions.

  • xsep

    • Default: 0.0d0

    • Example: xsep = 1.0d0

    Specifies the boundary positions between regions with different moving-average widths.

    This parameter is not used when nregion = 1, in which case xsep = 0.0d0 should be specified.

    For example, when dividing the domain into \(N\) regions, \(N-1\) boundary values must be specified.

  • npoint

    • Default: None

    • Example: npoint = 3 5

    Specifies the number of data points used for the moving average in each region.

    An odd number must be specified.

    In the above example,

    • Region 1: 3-point moving average

    • Region 2: 5-point moving average

    are applied.

  • include_zero

    • Default: .true.

    • Example: include_zero = .false.

    Specifies whether to preserve the value at the origin after smoothing.

    If .true., the value at the origin is kept unchanged from the input data.

    • Column 2: Function value

Average Function (average_function, avef)

In this analysis mode, one-dimensional functions stored in multiple input files are averaged. For example, let \(f_{i}\left(x\right)\) be the function stored in the \(i\)-th file and let \(N\) be the total number of files. The averaged function \(f_{\mathrm{ave}}\left(x\right)\) is given by

(72)\[\begin{align} f_{\mathrm{ave}}\left(x\right) = \dfrac{1}{N}\sum_{i=1}^{N}f_{i}\left(x\right), \end{align}\]
